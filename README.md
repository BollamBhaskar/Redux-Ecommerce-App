# Redux E-commerce Application

## 📌 Description
This project is a simple E-commerce web application built using React and Redux.  
The application demonstrates how Redux can be used for global state management in a real-world web application.

Users can browse products, add items to the cart, and manage their cart using Redux state.

---

## 🚀 Features
- Product listing
- Add items to cart
- Remove items from cart
- Increase or decrease product quantity
- Redux state management
- Responsive user interface

---

## 🛠 Tech Stack
- React.js
- Redux Toolkit
- JavaScript
- HTML
- CSS

---

## 📂 Project Structure
- Components for UI
- Redux store for global state management
- Product listing components
- Cart management functionality
